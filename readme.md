# Día del Amor y la Amistad – Proyecto Final
Este proyecto consiste en el desarrollo de una página web básica utilizando HTML y CSS, publicada mediante GitHub Pages.
El tema elegido es el 14 de febrero – Día del Amor y la Amistad. La página presenta información sobre la fecha, una imagen temática, un enlace externo y una encuesta interactiva mediante formulario.

## Descripción
La página incluye:
* Estructura semántica correcta (`header`, `main`, `section`,`footer`)
* Títulos jerárquicos (`h1`, `h2`)
* Párrafos informativos
* Imagen temática con atributo `alt`
* Enlace externo a playlist romántica
* Encuesta interactiva mediante formulario (`form`, `input`, `label`, `button`)
* Estilos CSS básicos con diseño centrado y colores temáticos

## Tecnologías utilizadas
